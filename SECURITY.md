# Security Policy

## Reporting a vulnerability

Please report security issues privately by opening a GitHub security advisory for this repository.

Include:
- affected version/commit
- impact and attack preconditions
- reproduction steps
- suggested mitigation (if known)

Do not open public issues for active vulnerabilities.

## Scope

This project interacts with USB HID hardware and desktop capture paths. Vulnerabilities related to device I/O, privilege boundaries, and unsafe defaults are in scope.
